copyThis.js

copyThis.js is here to copy Text in Google Chrome in Javascript.

Use

To use copyThis.js, you just have to install the Chrome Extension and include copyThis.js on your webpage. The chrome Extension will be avaivable in the Chrome Web Store as soon as possible.
<script src="https://raw.github.com/dervondenbergen/copyThis/master/copyThis.min.js"></script> Just click on the code to copy it.

Examples

Example using copyThis.js:

Javascript:

var button = document.getElementById("button");
var input = document.getElementById("input");
button.addEventListener("click", function(){
  copyThis(input.value);
});
